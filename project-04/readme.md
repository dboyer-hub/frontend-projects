# HTML Audio & Video Player

## Description
A simple HTML project created to practice embedding and controlling audio and video media using native HTML elements.

## Concepts Used
- Semantic HTML structure
- `<video>` element with controls
- `<audio>` element with controls and looping
- Media sources and fallback text
- Basic responsive considerations with viewport meta tag

## Features
- Embedded video player with playback controls
- Embedded audio player with loop functionality
- Graceful fallback messages for unsupported browsers
- Clean and minimal layout

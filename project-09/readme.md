# Quincy’s Tips for Getting a Developer Job

## Description
A simple HTML project created to practice using semantic text elements while presenting motivational content about starting a career in software development.

## Concepts Used
- Semantic HTML structure
- Inline and block quotations (`<q>`, `<blockquote>`)
- Proper use of citation attributes (`cite`, `<cite>`)
- Headings and content hierarchy
- Grouping content with `<main>` and `<section>`

## Features
- Introductory quote with inline citation
- Multiple sections highlighting key career concepts
- Block quotes for longer excerpts
- Clear structure for readability and meaning
- Proper document setup with meta tags

## Source
Content adapted from Quincy Larson’s *How to Learn to Code and Get a Developer Job* as part of my web development learning journey with freeCodeCamp.

# Video Compilation Page

## Description
A static HTML video compilation page created to practice structuring content using semantic HTML and embedding multiple external videos with iframes.

## Concepts Used
- Semantic HTML elements (`main`, `section`, headings, paragraphs)
- `<iframe>` element for embedding YouTube videos
- iframe attributes (`src`, `width`, `height`, `title`)
- Accessibility considerations for embedded media
- Content organization and hierarchy

## Features
- Main page title and introduction
- Multiple video sections with headings and descriptions
- Embedded YouTube videos with fullscreen support
- Clear separation of content using semantic sections

# Description List – Web Browsers

## Description
A simple HTML project created to practice using semantic description lists to present structured information clearly and meaningfully.

## Concepts Used
- Semantic HTML structure
- Description lists (`<dl>`, `<dt>`, `<dd>`)
- Headings and content hierarchy
- Clear pairing of terms and definitions

## Features
- List of major web browsers
- Each browser paired with a concise description
- Clean and readable layout using semantic elements
- Proper document structure with meta tags

# iFrame Video Display

## Description
A simple HTML project created to practice embedding external video content using the `<iframe>` element.

## Concepts Used
- HTML `<iframe>` element
- Embedding external content (YouTube video)
- iframe attributes (`src`, `width`, `height`)
- Accessibility considerations
- Security and privacy-related attributes

## Features
- Embedded YouTube video player
- Responsive-ready viewport setup
- Fullscreen video support
- Proper permission handling for embedded media
